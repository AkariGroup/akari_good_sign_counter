# akari_good_sign_counter
いいねをカウントするアプリです。  
AKARIに向けていいねポーズをすると、1いいねとしてカウントされます。  

## セットアップ
1. submoduleのclone  
    `git submodule update --init --recursive`  
2. 仮想環境の作成  
    `python -m venv venv`  
    `source venv/bin/activate`  
    `pip install -r requirements.txt`

## アプリの実行
`python3 main.py`
